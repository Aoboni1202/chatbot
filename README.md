#  Funny Chatbot Android App

## Overview

This is a simple and interactive **AI-powered chatbot Android application** that can perform basic conversations, mathematical calculations, tell jokes, and engage in casual gossip-style interactions.

The chatbot uses an external AI API to generate dynamic and intelligent responses, making the app fun and engaging for users.

---

##  Features

###  Smart Chat System
- Natural and interactive conversations  
- Casual chatting and storytelling responses  
- Fun and engaging dialogue experience  

###  Basic Math Support
- Simple arithmetic calculations  
- Helps solve basic mathematical problems  

###  Entertainment Mode
- Generates jokes for users  
- Gossip-style fun responses  
- Light and entertaining conversation flow  

###  AI Integration
- Connected with external AI chatbot API  
- Real-time response generation  
- Intelligent conversation handling  

---

##  Technologies Used

### Android Development
- Java  
- XML (User Interface Design)  

###  AI Integration
- Brainshop AI API  
- REST API (HTTP Requests)  

---

##  Project Purpose

The purpose of this project is to:
- Build a lightweight and fun chatbot application  
- Learn API integration in Android development  
- Demonstrate basic AI-based conversational system  
- Provide entertainment through interactive chatting  

---

## How It Works

1. User enters a message in the chat screen  
2. Message is sent to Brainshop AI API  
3. API processes the input and generates a response  
4. Response is displayed in the chat interface
